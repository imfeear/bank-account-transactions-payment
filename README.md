# JalaBank Project

This repository contains the source code for the JalaBank project, a web development project simulating a banking system.

## Modules

The project is divided into the following modules:

- **Accounts and Transactions**: Handles account creation, balance inquiries, deposits, withdrawals, transfers, and statement generation.
- **Payments**: Processes bill payments, manual payments, payment scheduling, and payment history.

## Requirements Analysis

Detailed requirements analysis for each module can be found in the respective module directories. A brief overview is provided below:

### Accounts and Transactions

- **Functional Requirements**: Account opening, balance inquiry, transactions (deposit, withdrawal, transfer), statement generation, authentication, authorization, user data update, notifications.
- **Non-Functional Requirements**: Security, performance, scalability, reliability, usability, maintainability, compatibility.

### Payments

- **Functional Requirements**: Bill payment, manual payments, payment scheduling, payment history.
- **Non-Functional Requirements**: Security, performance, reliability, scalability, compliance.
